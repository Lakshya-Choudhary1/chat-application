# 💬 Real-Time Chat Application

A modern **full-stack real-time chat application** built with React, Node.js, and MongoDB.
This project focuses on **secure authentication, instant messaging, and a smooth user experience** with modern web technologies.

## ✨ Features

* 🔐 **Custom JWT Authentication** – Secure login and signup without third-party auth
* ⚡ **Real-time Messaging** using Socket.IO
* 🟢 **Online / Offline Presence Indicators**
* 🔔 **Typing & Notification Sounds** with toggle option
* 📨 **Automated Welcome Emails** using Resend
* 🗂️ **Image Upload Support** via Cloudinary
* 🧰 **REST API** built with Node.js and Express.js
* 🧱 **Database Persistence** with MongoDB
* 🚦 **API Rate Limiting** powered by Arcjet
* 🎨 **Modern UI** built with React, Tailwind CSS, and DaisyUI
* 🧠 **Global State Management** using Zustand
* 🚀 **Deployment Ready** and free-tier friendly
